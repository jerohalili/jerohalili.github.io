### The Challenge
The organization relied on manual, paper-based processes for student enrollment and record-keeping, leading to data redundancy and slow processing times.

### The Solution
I developed a comprehensive management suite using the **MERN stack** (MongoDB, Express, React, Node.js). This included a centralized database to handle student records and a streamlined admin dashboard for real-time data visualization.

### Key Features
* **Automated Enrollment:** Reducing manual entry time by 70%.
* **Centralized Database:** Ensuring data integrity across all administrative departments.
* **Role-Based Access:** Secure login for students, faculty, and administrators.
