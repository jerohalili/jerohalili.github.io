## Short Introduction

This portfolio website serves as my central digital hub that showcases my work as a **Computer Scientist, Full-Stack Developer, AI Engineer, Game Developer, Marketer, and Graphic Designer**.

It was built to solve a real career problem: recruiters and clients often spend less than a minute evaluating candidates. Instead of forcing them to search through multiple platforms, this website consolidates my projects, certifications, technical skills, and professional experience into one fast and modern platform.

**Core Philosophy:** *Turning workflows into code.*

The portfolio emphasizes projects that solve real operational problems through automation, software systems, AI integration, and scalable web development.

## Development Process (How It Was Built and Why)

### Why I Built It

Traditional resumes are static and limited.

I needed a platform that could:

- Showcase technical projects visually
- Demonstrate real-world business impact
- Improve personal branding
- Increase recruiter conversion rates
- Differentiate myself from generic portfolios

From a marketing perspective:
A portfolio acts like a **sales funnel**:

Visitor → Interest → Trust → Contact

The site was designed to reduce friction and make contacting me easy.

---

### How It Was Built

### 1. Research Phase
I analyzed:
- Developer portfolios
- Recruiter behavior
- UX patterns
- Personal branding strategies

---

### 2. Planning Phase
I mapped:

- Homepage structure
- Project hierarchy
- Skills sections
- Certifications
- Contact funnels

---

### 3. Development Phase
Built using Astro because:

- Faster than traditional frameworks for static content
- Better SEO
- Lower hosting costs
- Easier long-term maintenance

Tailwind CSS was used for:
- Rapid styling
- Consistent design system
- Faster iteration

---

### 4. Optimization Phase
Improved:

- Mobile responsiveness
- Page speed
- Accessibility
- Theme persistence
- User navigation

---

### 5. Deployment Phase
Published through GitHub Pages with custom domain support.

## How to Run the Project

### Clone Repository
```bash
git clone https://github.com/jerohalili/jerohalili.github.io.git
```

### Enter Project Folder
```bash
cd jerohalili.github.io
```

### Install Dependencies
```bash
npm install
```

### Start Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
