Situation

The problem worth solving

A SaaS company was producing content manually — a slow, expensive, and fundamentally unscalable process. Their social presence was inconsistent, which directly hurt algorithmic reach and audience growth. The team had ideas but no bandwidth.

The core insight: content volume compounds. More posts mean more impressions, more data, more targeting opportunity. The constraint wasn't ideas — it was the cost and time of production.

Task

What I was hired to build

Design and build a fully automated marketing pipeline that could:

  • Generate on-brand marketing copy using AI with no human writing time
  • Produce matching visual assets automatically
  • Schedule and publish content across social platforms on a set cadence
  • Operate continuously with an optional human approval loop
  • Run locally to preserve data privacy and eliminate API costs
Action

How I built it

01

Local LLM for copy generation

Deployed LLaMA locally via Ollama, crafted a structured prompt system for generating varied, on-brand post copy — headlines, captions, and hashtags tailored to the company's tone. Local inference means zero per-token costs and full data privacy.

LLaMA · Ollama · Prompt Engineering
02

Automated image generation

Integrated ComfyUI with Z-Image Turbo for fast, high-quality image generation. Each post triggers a paired image creation workflow — prompt derived from copy context. The output is consistent, on-brand visual content without a designer in the loop.

ComfyUI · Z-Image Turbo · SDXL
03

n8n orchestration & scheduling

Built the entire workflow in n8n — the orchestration layer that triggers LLM generation, passes output to image generation, assembles the final post package, and publishes via platform APIs on a defined schedule. Added an optional approval node for human review before publish.

n8n · Workflow Automation · API Orchestration
Result

What changed

📈

Content volume exploded

Posting frequency increased dramatically — moving from occasional manual posts to a consistent, high-cadence schedule. Algorithms reward consistency; this system delivered it reliably.

💰

Cost dropped to near zero

With local inference and local image generation, the ongoing marginal cost per post was effectively zero — no API fees, no designer hours, no copywriter time.

🔄

Compounding growth flywheel

More content → more impressions → more follower data → better targeting. The system created a self-reinforcing loop that improves over time without additional human effort.
